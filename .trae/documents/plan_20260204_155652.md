# 系统数据实时化与可视化升级计划

## 目标
将系统面板中的所有模拟/虚假数据替换为真实运行数据，确保用户看到的每一个指标都有据可循。

## 实施方案

### 1. 数据源架构升级 (`src/store/useGameStore.ts`)
我们需要在全局状态管理中新增以下字段来支撑前端显示：
- **`startTime`**: 记录应用启动时间，用于计算精确的“运行时间”。
- **`requestCount`**: 记录 API 请求总数，每次 `fetch` 成功或失败都自增。
- **`logs`**: 一个字符串数组队列（Max 50），用于存储真实的系统日志（如“正在更新 CS2...”、“网络请求失败”等）。
- **`systemStatus`**: 细化状态枚举（`idle` | `updating` | `error` | `offline`）。
- **`totalPlayers`**: 所有监控游戏在线人数的总和，作为“社区状态”或“当前监控热度”的真实数据。

### 2. 组件改造

#### A. 系统状态面板 (`src/components/SidePanels/StatusPanel.tsx`)
- **控制台输出 (Console Output)**:
  - **现状**: 随机生成的假日志。
  - **改造**: 绑定 `useGameStore.logs`，显示真实的 API 请求记录、错误信息和更新状态。
- **运行时间 (Runtime)**:
  - **现状**: 静态或假计时。
  - **改造**: 基于 `startTime` 使用 `setInterval` 每秒更新显示 `Date.now() - startTime`。
- **请求总数 (Total Requests)**:
  - **现状**: 静态数字。
  - **改造**: 绑定 `useGameStore.requestCount`。
- **API 配额 / 内存占用**:
  - **改造**: 
    - **API 配额**: 既然无法获取 Steam 真实配额，改为显示 **“当前并发数”** 或 **“最近一分钟请求数”** (RPM)，这是真实可测的。
    - **内存占用**: 浏览器环境无法直接获取准确的系统内存，改为显示 **“监控游戏数量”** (占用资源的直接指标) 或 **“缓存大小”**。

#### B. 网络仪表盘 (`src/components/NetworkDashboard.tsx`)
- **延迟 (Ping)**:
  - **现状**: 模拟请求 npmmirror。
  - **改造**: 保持请求真实 CDN (如 Cloudflare 或 Steam 静态资源) 的逻辑，但确保它反映的是真实的网络连通性。
- **上传/下载速度**:
  - **现状**: 随机数。
  - **改造**: 浏览器无法获取系统级网速。改为显示 **“数据吞吐量”** (Data Throughput)，即本次会话中从 Steam API 实际接收到的 JSON 数据大小（估算值）。

#### C. 趋势与负载面板 (`src/components/SidePanels/TrendsPanel.tsx`)
- **全球趋势 (Global Trends)**:
  - **现状**: 硬编码的游戏列表 (Palworld 等)。
  - **改造**: 改为 **“监控列表趋势”**。显示用户监控列表中，在线人数变化最大的前 5 款游戏（基于 `playerCount` 变化）。
- **服务器负载 (Server Load)**:
  - **现状**: 假动画。
  - **改造**: 改为 **“API 响应延迟”**。显示最近一次 Steam API 请求的耗时 (Latency)。
- **社区状态 (Community Status)**:
  - **现状**: 假的 24.5M 在线。
  - **改造**: 改为 **“监控总在线”**。实时计算用户监控的所有游戏当前的在线人数总和。

## 执行步骤
1.  **修改 Store**: 在 `useGameStore` 中添加 `startTime`, `requestCount`, `logs`, `totalDataTransferred` 等状态，并提供 `addLog`, `incrementRequest` 等 Action。
2.  **埋点**: 在 `steamApi.ts` 和 `App.tsx` 的数据获取逻辑中，插入日志记录和计数代码。
3.  **重构组件**: 依次修改 `StatusPanel`, `NetworkDashboard`, `TrendsPanel`，将硬编码数据替换为 Store 中的动态数据。
